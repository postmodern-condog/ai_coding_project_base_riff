# Contributing

Thanks for your interest in contributing to the AI Coding Project Toolkit.

## Quick Start

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Make your changes
4. Run `npm run lint` to check formatting
5. Commit with a clear message
6. Open a pull request

## Guidelines

- **Read [AGENTS.md](AGENTS.md)** for detailed workflow rules specific to this codebase
- **Keep changes focused** — prefer small, targeted edits over large rewrites
- **Preserve compatibility** — existing command names and workflows should not break without strong reason
- **Treat prompts as code** — update related docs when behavior or outputs change

## What to Contribute

- Bug fixes and improvements to existing commands/skills
- Documentation improvements
- New commands or skills that fit the existing patterns
- Tests and validation improvements

## Questions?

Open an issue for discussion before starting large changes.
